## 📌 项目简介

本固件专为竞赛三轴机械臂设计，采用 **STM32H723VGT6**（带 FPU）作为主控，实现：

- **三路步进电机闭环伺服控制**：ZDT X42 闭环步进电机，自带增量式编码器
- **双环 PID 控制**：速度环（内环）+ 位置环（外环），控制频率 1kHz，支持脉冲频率输出
- **梯形轨迹规划**：平滑位置/速度/加速度曲线，减少机械冲击
- **姿态解算（KDL 库）**：正逆运动学解算，支持关节空间/笛卡尔空间轨迹
- **FreeRTOS 任务调度**：隔离控制、规划、通信等实时任务
- **FPU 加速**：充分利用 STM32H7 浮点性能，满足高计算密度需求

项目代码完全开源，提供工程文件（STM32CubeMX / Keil），可直接编译烧录。

### 关键模块说明

#### 1. 电机驱动 + 编码器读取
- 使用通用定时器输出 **脉冲信号**（频率决定速度，个数决定位置）与 **方向电平**，连接步进电机驱动器的 PUL/DIR 接口。
- 编码器信号接入定时器的编码器接口，硬件完成四倍频计数，分辨率 4000 脉冲/转，方向自动更新。

#### 3. 梯形轨迹规划
- 在线生成梯形速度曲线，包含加速、匀速、减速三段，保证运动平滑，避免冲击。
- 每一控制周期输出目标位置、速度、加速度，供位置环跟踪。

#### 4. KDL 运动学解算
- 基于标准 D‑H 参数建立三轴机械臂模型，提供正运动学、逆运动学接口。
- 轨迹点可由关节空间或笛卡尔空间给出，逆解结果转换为各轴目标位置。

#### 5. FreeRTOS 任务调度
- **控制任务**：0.1kHz 严格定时，执行脉冲更新。
- **通信任务**：负责接收指令、解析协议、回传状态。
- 任务间通过队列和信号量同步，保证实时性。

#### 6. UART 上位机下位机通信
- AXYZxyzB
- A包头 B包尾 
- XYZxyz均只填非负值
- 例:
- A100000B 末端向前
- A100100B 末端向后
- A010000B 末端向左
- A010010B 末端向右
- A001000B 末端向上
- A001001B 末端向下
