## 📌 项目简介

本固件专为竞赛三轴机械臂设计，采用 **STM32H723VGT6**（带 FPU）作为主控，实现：

- **三路电机独立伺服控制**：YAHBOOM 520 型直流减速电机 + AB 相霍尔编码器
- **双环 PID 控制**：速度环（内环）+ 位置环（外环），控制频率 1kHz
- **梯形轨迹规划**：平滑位置/速度/加速度曲线，减少机械冲击
- **姿态解算（KDL 库）**：正逆运动学解算，支持关节空间/笛卡尔空间轨迹
- **FreeRTOS 任务调度**：隔离控制、规划、通信等实时任务
- **FPU 加速**：充分利用 STM32H7 浮点性能，满足高计算密度需求

项目代码完全开源，提供工程文件（STM32CubeIDE / Keil），可直接编译烧录。

## ✨ 主要特性

| 模块             | 实现方式                                               | 关键参数                                 |
| ---------------- | ------------------------------------------------------ | ---------------------------------------- |
| **电机驱动**     | PWM（50kHz） + 双路编码器输入（TIM 编码器模式）        | 死区补偿、方向控制                        |
| **编码器反馈**   | AB 相霍尔编码器（YAHBOOM 520 型）                      | 分辨率：4000 线 × 4 倍频 = 16000 脉冲/转 |
| **PID 控制**     | 增量式 PID + 限幅积分分离                              | 速度环 1kHz，位置环 1kHz                  |
| **轨迹规划**     | 梯形速度曲线（Trapz）                                  | 最大速度 / 加速度 / 加加速度可配置        |
| **运动学**       | KDL 库移植（C++ 封装，正逆解）                         | 标准 D-H 参数（3R 构型）                  |
| **通信接口**     | USART（协议可配）+ CAN 扩展预留                        | 115200 bps，二进制指令帧                  |
| **任务调度**     | FreeRTOS v10.5.1                                      | 4 个用户任务 + 空闲任务                   |

## 🧱 硬件平台

| 组件               | 型号 / 规格                                         |
| ------------------ | --------------------------------------------------- |
| **主控 MCU**       | STM32H723VGT6（Cortex‑M7 @ 550MHz，双精度 FPU）    |
| **电机** ×3        | YAHBOOM 520 型直流减速电机 + AB 相霍尔编码器        |
| **电机驱动** ×3    | TB6612 / DRV8870（支持 PWM 调速 + 方向信号）       |
| **电源**           | 12V / 5A（电机供电）+ 3.3V（主控供电）             |
| **编码器接口**     | TIM2, TIM3, TIM8（编码器模式）                     |
| **调试接口**       | SWD（JTAG）+ 串口打印（USART1）                    |
| **其他外设**       | LED 指示灯，按键，I2C（预留扩展）                  |

> **接线简表**（参考）
> | 电机编号 | PWM 引脚       | 编码器 A/B 引脚   |
> |----------|----------------|-------------------|
> | 关节1    | PA8, PB13      | PA0 (TIM2_CH1/CH2)|
> | 关节2    | PB6, PB7       | PA1 (TIM5_CH1/CH2)|
> | 关节3    | PC6, PC7       | PB0 (TIM3_CH3/CH4)|
